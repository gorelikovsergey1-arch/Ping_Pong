# Ping_Pong
Описание проекта

Этот репозиторий содержит код прототипа классической игры «Пинг-понг». Задача проекта — реализовать базовый функционал игры без полного интерфейса, сосредоточившись на ключевых механиках.
Основные функции

В рамках проекта реализовано:

    Автоматическое перемещение мяча по сцене — мяч движется с постоянной скоростью, отскакивая от краёв экрана и ракеток.
    Две ракетки, управляемые с клавиатуры — каждый игрок контролирует свою ракетку с помощью клавиш (назначение клавиш можно настроить в коде).
    Условие проигрыша — игрок проигрывает, если пропускает мяч (мяч уходит за пределы его стороны экрана).
    Управление

    Игрок 1 (левая ракетка):
        Вверх — клавиша W.
        Вниз — клавиша S.
    Игрок 2 (правая ракетка):
        Вверх — клавиша ↑ (стрелка вверх).
        Вниз — клавиша ↓ (стрелка вниз).

Примечание: назначение клавиш можно изменить в коде, отредактировав соответствующие участки.
Возможности для расширения

Проект оставляет пространство для улучшений и дополнений. Примеры возможных доработок:

    добавление системы подсчёта очков;
    реализация смены сторон после определённого количества очков;
    добавление анимации при столкновении мяча с ракеткой;
    внедрение звуковых эффектов (удары, проигрыш);
    изменение дизайна (цвета, формы мяча и ракеток);
    добавление меню старта и настроек;
    реализация одиночного режима (компьютерный соперник).
